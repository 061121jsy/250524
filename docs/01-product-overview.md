# 01. Product Overview

## 목표

독할매카세는 사주, 운세, 궁합을 한 화면에서 읽고 저장하는 테스트 플랫폼입니다.

## 제품 방향

- 일반 회원가입과 카카오 소셜 로그인을 모두 지원합니다.
- 카테고리는 연애, 재물, 직업, 궁합으로 나눕니다.
- 가이드 캐릭터는 달솔, 금나래, 초연으로 통일합니다.
- 결과 저장과 이미지 관리는 Supabase Storage, 데이터는 Supabase Postgres에 둡니다.

## 1차 범위

- 로그인 허브
- 로그인 화면
- 회원가입 화면
- 정통 사주 화면
- 타로 화면
- 사주 입력과 결과 화면
- 이미지 기반 비주얼 영역
- 마이페이지
