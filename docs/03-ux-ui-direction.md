# 03. UX/UI Direction

## 시각 방향

- 검정, 짙은 갈색, 금색을 중심으로 구성
- 전통 점술 무드와 프리미엄 카드 레이아웃 결합
- 입력 전에 분위기를 먼저 보여주는 히어로 화면 구성

## 화면 구성

- 상단: 브랜드 로고와 로그인 진입
- 중앙: 사주 입력 전의 분위기와 핵심 가치 제시
- 중단: 운세 카테고리 카드
- 하단: 저장 구조, AI 구조, 캐릭터 소개

## 캐릭터 이름

- 달솔: 연애 가이드
- 금나래: 재물 가이드
- 초연: 진로 가이드

## 디자인 원칙

- 모바일 우선
- 카드와 패널은 낮은 라운드
- 금색 포인트는 강조에만 사용
- 텍스트는 짧고 직접적으로 유지
